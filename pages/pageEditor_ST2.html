HomePage

Technical Description

The Home page was created using HTML5 and CSS3 with a focus on responsiveness

Key Technices:

  • Navigation Bar: Styled with flexbox, hover effects and an active page indicator.
  • Card Layout: Used CSS Grid/ Flexbox for a uniform tiles structure.
  • Responsive Design: Media quarries to ensure proper layout across devices.

Accessibility

Implimented alt attributes for images, proper contrast rastions and ensured key board navigation.
